# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and harassment-free experience for everyone.

## Standards

**Be respectful.** Disagreements are fine — insults are not.

**Be constructive.** Focus on the code and the ideas, not the person.

**Be patient.** Not everyone has the same background or experience level.

## Scope

This applies to all project spaces: GitHub issues, pull requests, and any project-related communication.

## Enforcement

Instances of abusive behavior may be reported to the project maintainers. All complaints will be reviewed and investigated.
