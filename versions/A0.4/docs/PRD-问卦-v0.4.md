## 1. Executive Summary

**问卦 v0.5** 新增姓名起卦模式 — 用户输入中文姓名，系统自动将每个字转换为繁体康熙字典笔画数，按梅花易数姓名占法生成终身卦，输出本卦/互卦/变卦及体用生克解读，判断名字对命主的吉凶助损。

---

## 2. Problem Statement

### 用户痛点

- **起名/改名需求普遍** — 很多人想知道自己的名字好不好，是否需要改名
- **笔画不会算** — 普通人不知道姓名起卦必须用繁体字笔画（康熙字典标准），简体笔画差异巨大（龙5→龍16）
- **姓名占是梅花易数的经典应用** — 邵雍原书中就有姓名占，这种体验是算卦工具不可或缺的

### 业务影响

- 姓名起卦是 research/07 就规划好的功能，研究文档 146 行完整覆盖算法 + 笔画规则 + 示例
- v0.2-v0.4 已交付铜钱、梅花三数、时间、生辰四种起卦方式，姓名是第五种也是最具社交传播力的

## 5. User Stories

### US-1: 姓名输入起卦（P0）

```
Feature: 姓名起卦

  Scenario: 用户输入单姓双名起卦
    Given 用户在姓名起卦模式
    When 输入"诸葛亮"并点击"起卦"
    Then 系统自动识别为单姓双名（诸+葛+亮）
    And 自动查繁体笔画: 诸15画/葛12画/亮9画
    And 上卦 = 15%8 = 7 → 艮☶
    And 下卦 = (12+9)%8 = 5 → 巽☴
    And 动爻 = (15+12+9)%6 = 0 → 第6爻
    And 本卦 = 山风蛊䷑
    And 变卦 = 地风升䷭
    And 展示完整姓名卦结果

  Scenario: 复姓支持
    Given 用户输入"欧阳修"
    When 系统识别为复姓单名
    Then 上卦 = (欧+阳)笔画%8，下卦 = 修笔画%8

  Scenario: 单字名
    Given 用户输入"王明"
    Then 上卦=王4%8=4震☳, 下卦=明8%8=0坤☷, 动爻=12%6=0→6

  Acceptance Criteria:
  - [ ] 支持单姓单名/单姓双名/复姓单名/复姓双名四种结构
  - [ ] 每个字的繁体笔画正确显示给用户
  - [ ] 用户可手动修正笔画数（如果内置表不准确）
```

### US-2: 简繁笔画转换（P0）

```
Feature: 康熙字典笔画查表

  Scenario: 简体字自动匹配繁体笔画
    Given 用户输入简体"龙"
    When 系统查 STROKE_MAP['龙']
    Then 返回繁体笔画 5（简体）→ 16（康熙字典）
    And 界面显示"龙 → 龍（16画）"

  Scenario: 偏旁部首笔画
    Given 用户输入含"扌"的字（如"打"）
    Then 扌(提手旁) → 算作"手"4画

  Scenario: 未收录字兜底
    Given 用户输入生僻字（STROKE_MAP 中不存在）
    Then 系统用 Unicode CJK 粗略估算并提示用户确认
    And 允许用户手动覆盖笔画数

  Acceptance Criteria:
  - [ ] STROKE_MAP 含 ≥3000 条繁体笔画映射
  - [ ] 偏旁部首：扌4/氵4/忄4/王5/礻5/衤6/阝7-8/讠7
  - [ ] 未收录字提示用户手动输入
```

### US-3: 姓名卦结果渲染（P0）

```
Feature: 姓名卦解读

  Scenario: 完整结果展示
    Given 姓名起卦完成
    When 渲染结果
    Then 展示:
      - 姓名 + 每字繁体笔画明细
      - 本卦卡片 + 互卦卡片 + 变卦卡片（复用梅花引擎）
      - 体用生克卡片 + 命格解读
      - 姓名点评："用生体·大吉 ✅ — 这名字旺你，对命主有助益"
      - "姓名对命主：助 ××" 或 "姓名对命主：克 ××"

  Scenario: 各等级命名评价
    When 用生体(level=5)
    Then 评价 = "大吉·旺命名 — 名字对命主有强大助益"
    When 用克体(level=1)
    Then 评价 = "凶·阻命名 — 名字给命主带来阻力，建议改名"

  Acceptance Criteria:
  - [ ] 笔画明细醒目展示（每个字→繁体→画数）
  - [ ] 命名评价对应5级体用生克
```

## 8. Technical Considerations

### 8.1 笔画数据来源

康熙字典笔画需从权威来源采集。3000+ 常用字覆盖日常姓名需要，可参考：
- 在线康熙字典 API 的离线数据快照
- Unicode Han 数据库 `kTotalStrokes` 字段
- MIT 协议开源笔画库

### 8.2 姓名结构识别

```javascript
// 复姓表（常见50个）
const COMPOUND_SURNAMES = ['欧阳','司马','上官','诸葛','夏侯','慕容',...];

function parseName(name) {
    if (name.length === 2) return { type:'single-single', parts:[name[0], name[1]] };
    if (name.length === 3) {
        const prefix = name.substring(0,2);
        if (COMPOUND_SURNAMES.includes(prefix))
            return { type:'compound-single', parts: [prefix, name[2]] };
        return { type:'single-double', parts: [name[0], name.substring(1,3)] };
    }
    if (name.length === 4) {
        const prefix = name.substring(0,2);
        if (COMPOUND_SURNAMES.includes(prefix))
            return { type:'compound-double', parts: [prefix, name.substring(2,4)] };
        return { type:'compound-double', parts: [name.substring(0,2), name.substring(2,4)] };
    }
    // 5字以上取前一半/后一半
}
```

## 10. Out of Scope

- ❌ 全量 50000+ 汉字康熙笔画（体积过大）→ 仅 3000 常用字
- ❌ 八字/三才五格/五格三才数理结合姓名卦 → 远期
- ❌ 姓名改名建议（自动推荐吉名）→ 远期
- ❌ 日文/韩文汉字笔画 → 远期
- ❌ 姓名卦的流年运势 → 远期
