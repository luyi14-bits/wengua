# 解卦规则

## 核心法则：朱熹变占法

宋代大儒朱熹在《易学启蒙》中归纳了变卦解卦的完整方法，这是最通行的解卦标准。

### 按动爻数量取用

| 动爻数 | 解卦方法 | 说明 |
|--------|----------|------|
| **0** (六爻全静) | 看本卦**卦辞** | 事态平稳，无变化趋势 |
| **1** | 看本卦**该动爻爻辞** | 变动集中于一爻，该爻为关键 |
| **2** | 看本卦**两动爻爻辞**，以**上爻为主** | 两处变动，以上方爻辞为重 |
| **3** | 看本卦卦辞 + 变卦卦辞，以**本卦为主** | 三爻变动较大，兼看两卦 |
| **4** | 看变卦中**两个不变爻**的爻辞，以**下爻为主** | 多数已变，看不变者 |
| **5** | 看变卦中**唯一不变爻**的爻辞 | 只一爻未变，该爻定吉凶 |
| **6** (六爻全动) | 看变卦**卦辞** | 乾看"用九"，坤看"用六"，其余看卦辞 |

### 乾、坤特殊规则

- **乾卦六爻全动** → 用"用九：见群龙无首，吉"
- **坤卦六爻全动** → 用"用六：利永贞"

### 解卦层级

1. **卦辞** — 全卦总体判断（宏观层面）
2. **彖辞** — 卦辞的哲学解释（中观层面）
3. **大象** — 从卦象引申的处世道理（修身层面）
4. **爻辞** — 具体时位的吉凶指示（微观层面）
5. **小象** — 爻辞的补充说明

### 本玩具的解卦实现

简化处理，分两级展示：

**第一级（概述）**：
- 本卦卦名 + 卦符 + 卦辞
- 如有变卦，展示变卦卦名 + 卦符 + 卦辞
- 动爻标注

**第二级（详解）**：
- 按朱熹变占法自动选择对应爻辞/卦辞展开
- 六爻逐条展示，动爻高亮

### 动爻数 → 展示规则

```javascript
function getInterpretationFocus(benGua, bianGua, changingLines) {
    const count = changingLines.length;
    
    if (count === 0) return { type: 'guaci', gua: benGua };
    if (count === 1) return { type: 'yaoci', gua: benGua, line: changingLines[0] };
    if (count === 2) return { type: 'yaoci', gua: benGua, line: Math.max(...changingLines), secondary: Math.min(...changingLines) };
    if (count === 3) return { type: 'guaci', gua: benGua, secondary: bianGua };
    if (count === 4) return { type: 'yaoci', gua: bianGua, line: unchangedLines[0] };
    if (count === 5) return { type: 'yaoci', gua: bianGua, line: unchangedLines[0] };
    if (count === 6) {
        if (benGua.id === 1) return { type: 'yongjiu', gua: benGua };  // 乾用九
        if (benGua.id === 2) return { type: 'yongliu', gua: benGua };  // 坤用六
        return { type: 'guaci', gua: bianGua };
    }
}
```
