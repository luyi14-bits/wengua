# 梅花易数

> 北宋邵雍（康节先生）所创。其核心是"先天八卦数"（乾一兑二离三震四巽五坎六艮七坤八），
> 将万物转化为数字，由数起卦。与铜钱掷爻法完全不同。

---

## 一、先天八卦数 ⚠️ 与二进制序不同！

铜钱法用的是伏羲先天二进制序（坤0→乾63），而梅花易数有自己的一套数字系统：

| 八卦 | 先天数 | 五行 | 自然 | 口诀 |
|------|--------|------|------|------|
| 乾 ☰ | **1** | 金 | 天 | 乾三连 |
| 兑 ☱ | **2** | 金 | 泽 | 兑上缺 |
| 离 ☲ | **3** | 火 | 火 | 离中虚 |
| 震 ☳ | **4** | 木 | 雷 | 震仰盂 |
| 巽 ☴ | **5** | 木 | 风 | 巽下断 |
| 坎 ☵ | **6** | 水 | 水 | 坎中满 |
| 艮 ☶ | **7** | 土 | 山 | 艮覆碗 |
| 坤 ☷ | **8** | 地 | 地 | 坤六断 |

> 记法：乾1→兑2→离3→震4→巽5→坎6→艮7→坤8，即先天八卦方位序。

## 二、起卦方法

### 2.1 三数起卦法（最通用，最适合程序实现）

用户随意报三个数字，或用时间、页码等来源的数字，三个数分别决定上卦、下卦、动爻：

```
上卦数 = 第一个数字 mod 8 (余0则取8=坤)
下卦数 = 第二个数字 mod 8 (余0则取8=坤)
动爻   = 第三个数字 mod 6 (余0则取6=上爻)
```

**示例**：用户输入 15, 9, 20
- 上卦：15 % 8 = 7 → 艮☶
- 下卦：9 % 8 = 1 → 乾☰
- 动爻：20 % 6 = 2 → 二爻动
- 本卦：山天大畜 ䷙
- 变卦：二爻(阳变阴) → 山火贲 ䷕

### 2.2 年月日时起卦法

取当前公历时间：

```
上卦数 = (年 + 月 + 日) mod 8
下卦数 = (年 + 月 + 日 + 时) mod 8
动爻   = (年 + 月 + 日 + 时) mod 6
```

### 2.3 数字来源创意玩法（未来可扩展）

| 来源 | 示例 | 说明 |
|------|------|------|
| 用户输入 | 随意报三个数 | 最直接 |
| 当前时间 | 年月日时自动取 | 即时起卦 |
| 手机号后三位 | 138...678 | 取后三数 |
| 随手翻书 | 页码 | 取页数 |
| 字数笔画 | 姓名 | 拆字算笔画 |

## 三、互卦（梅花易数特有）

互卦是本卦中间四爻重新组成的卦，揭示内在联系和发展过程的中间状态。

```
本卦六爻：  [上爻, 五爻, 四爻, 三爻, 二爻, 初爻]
                └──────┬──────┘
                     取中间四爻
              [五爻, 四爻, 三爻, 二爻]

上互卦(外互) = 五爻 + 四爻 + 三爻   (上卦，前三爻)
下互卦(内互) = 四爻 + 三爻 + 二爻   (下卦，后三爻)
```

> 注意：互卦也是三爻为一组，上互卦取 5-4-3 爻，下互卦取 4-3-2 爻。
> 4爻和3爻被上下互卦共用——这是重叠取爻法的关键。

**示例** — 本卦䷙山天大畜 (初爻LSB: 111001)

爻位从上到下：上九(1), 六五(0), 六四(0), 九三(1), 九二(1), 初九(1)
中间四爻取：六五(0), 六四(0), 九三(1), 九二(1)

- 上互卦 = 六五+六四+九三 → 001 → 震☳ (mod 先天数: 二进制001=1对应兑才对...需要小心)

实际上互卦用的是先天八卦数匹配。更准确地说：中间四爻的取法本质是看爻的阴阳组合形成新的三爻卦。但互卦也可以用先天数来查。

简化处理：直接用爻的阴阳组合生成新卦。

## 四、体卦与用卦

梅花易数解卦的核心概念：

| 概念 | 定义 | 含义 |
|------|------|------|
| **体卦** | 不含动爻的那个单卦（上卦或下卦） | 代表问卦人自己、主体 |
| **用卦** | 含动爻的那个单卦 | 代表外部环境、对方、客体 |
| **本卦** | 原始卦象 | 当前状态 |
| **互卦** | 中间四爻重组的卦 | 发展过程的中间环节 |
| **变卦** | 动爻翻转变换后的卦 | 最终结局 |

**动爻位置判断体用：**
- 动爻在 1/2/3 爻（下卦区域） → 体卦=上卦，用卦=下卦
- 动爻在 4/5/6 爻（上卦区域） → 体卦=下卦，用卦=上卦
- 多动爻 → 取第一个动爻位置判断

## 五、体用生克（五行判吉凶）

八卦配五行后，体卦和用卦之间产生生克关系：

| 关系 | 五行逻辑 | 吉凶 | 解读 |
|------|----------|------|------|
| **用生体** | 用卦五行生体卦五行 | **大吉** ✅ | 外部环境滋养我，诸事顺利 |
| **体用比和** | 同五行（如乾兑同金） | **吉** ✅ | 人境和谐，内外同心 |
| **体克用** | 体卦五行克用卦五行 | **小吉** ➡️ | 我能掌控局面，但需花力气 |
| **体生用** | 体卦五行生用卦五行 | **泄气** ⚠️ | 我在消耗自己滋养环境 |
| **用克体** | 用卦五行克体卦五行 | **凶** ❌ | 外部环境压制我，诸事不顺 |

### 五行生克速查

```
木生火, 火生土, 土生金, 金生水, 水生木  (相生)
木克土, 土克水, 水克火, 火克金, 金克木  (相克)
```

### 八卦五行速查

```
乾☰=金 | 兑☱=金 | 离☲=火 | 震☳=木
巽☴=木 | 坎☵=水 | 艮☶=土 | 坤☷=土
```

## 六、经典案例（邵雍《梅花易数》原书）

### 观梅占（梅花易数得名之作）

辰年十二月十七日申时，邵雍见二雀争枝坠地，以此起卦：

- 年5 + 月12 + 日17 = 34 → 34%8=2 → **兑☱ 上卦**
- 34 + 申时9 = 43 → 43%8=3 → **离☲ 下卦**
- 43%6=1 → **初爻动**

本卦：泽火革 ䷰ → 初爻变 → 变卦：泽山咸 ䷞，互见乾巽。

断：兑金为体，离火克之（用克体，凶），互中巽木又生离火（火更旺），兑为少女故女子被伤；但变卦艮土生兑金（体得生），故不致丧命。明晚果然有女子折花，惊坠伤股。

### 牡丹占

巳年三月十六日卯时：

- 年6+月3+日16=25 → 25%8=1 → **乾☰ 上卦**
- 25+卯时4=29 → 29%8=5 → **巽☴ 下卦**
- 29%6=5 → **五爻动**

本卦：天风姤 ䷫ → 变卦：火风鼎 ䷱，互见重乾。

断：巽木为体，乾金克之（用克体），互重乾，克体太过。断牡丹明日午时被马践毁。次日果然应验。

## 七、本玩具的实现方案

梅花易数模式与铜钱模式的**关键差异**：

| 对比项 | 铜钱掷爻法 | 梅花易数 |
|--------|-----------|----------|
| 数据来源 | 随机三枚铜钱×6 | 三个数字 |
| 八卦数字 | 二进制序 (0-7) | 先天数 (1-8) |
| 互卦 | 无 | 有（中间四爻重卦） |
| 体用 | 无 | 有（体卦/用卦+五行生克） |
| 动爻数 | 可能多个 | 仅一个 |
| 解读侧重 | 卦辞+爻辞释义 | 体用生克+卦辞爻辞 |

### 数据模型扩展

```javascript
// 梅花易数结果
{
    benGua: Gua,              // 本卦
    huGua: Gua,               // 互卦 ← 新增
    bianGua: Gua|null,        // 变卦
    dongYao: number,          // 动爻位置 (1-6) ← 只有一个
    tiGua: Trigram,           // 体卦 (单卦)
    yongGua: Trigram,         // 用卦 (单卦)
    shengKe: {                // 体用生克 ← 新增
        relation: '用生体'|'体用比和'|'体克用'|'体生用'|'用克体',
        judgment: '大吉'|'吉'|'小吉'|'泄气'|'凶',
        reasoning: string
    }
}
```
